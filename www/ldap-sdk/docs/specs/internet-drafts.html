LDAP Specifications Defined in Internet Drafts

This page provides a listing of a number of LDAP-related specifications that are defined in Internet Drafts. Whenever possible, the most recently-published revision of the draft is provided.

Note that some of these drafts are still in active development and may not be considered stable. Other drafts are expired without having ever being promoted to an RFC. However, some LDAP clients and/or servers may have implemented support for some of these drafts.
